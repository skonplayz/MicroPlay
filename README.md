# MicroPlay

MicroPlay is a modern, lightweight 2D game engine built for speed, flexibility, and extensibility.

No crashes. No lost progress. Your work is always safe with automatic project autosaving.

## Features

- Real-time visual editor  
- Automatic autosaving  
- Built-in physics system  
- Tag-based object system  
- Rule-based behavior engine  
- Full JavaScript modding support  
- Lightweight and fast  
- Electron-based cross-platform editor  
- Extensible mod architecture  
- Modern dark UI
